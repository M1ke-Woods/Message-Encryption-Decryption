# Message-Encryption-Decryption
Encrypt and Decrypt Messages
## Directions of Use
1. Run the Encryption file first. 
2. When prompted for an input, type the message that you want encrypted here and press enter when you're finished. 
3. Copy the list of numbers without the parentheses 
4. Clear and close the encryption file
5. Run the Decryption file
6. When prompted, paste the list of numbers and press enter
7. Congratulations! You have now encrypted and decrypted your first message.
